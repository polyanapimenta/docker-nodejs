## Pré-requisitos: 
Ter o Docker e Docker Compose instalado
  - `docker -v`
  - `docker-compose -v`

## Para dar START na aplicação:
  - `cd docker-nodejs`
  - `docker-compose up --build`

A aplicação estará disponível em:
  http://localhost:3000
